## Variant: 気づき宇宙 Premium Prototype

### Design stance
「200万〜300万円級のデザイン/操作感」を目標に、気づきが星として浮かび、クリックすると星座・関連気づき・AIガイドが立ち上がる高品質プロトタイプ。

### Key choices
- Layout: 宇宙キャンバス + 上部モード切替 + 詳細パネル + AIガイド + テーマフィルタ + ミニマップ
- Typography: 日本語の読みやすさを保ちつつ、プレミアム感のある密度
- Color: 深いネイビー、金、アクア、グリーン、ローズ、バイオレット
- Interaction: カードクリックで関連星がハイライト、星座線を再描画、詳細パネル更新
- Motion: 星のゆっくりした流れ、光のパルス、カードの浮遊感

### Trade-offs
- Strong at: デモ映え、商品価値の伝達、探索体験、AI編集者感
- Weak at: 本番運用にはパフォーマンス/アクセシビリティ/スマホ最適化の追加検証が必要

### Best for
- Nobyさんが講座主や協力者に「こういう世界観にしたい」と見せるプレミアムデモ
- ここから実MVPの公開ページに組み込むための方向性決定
