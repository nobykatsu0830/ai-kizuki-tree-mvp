## Variant: 気づき宇宙

### Design stance
Google Earth/星座/探索感を強め、「つい見たくなる」発見型UIにする。

### Key choices
- Layout: 宇宙空間にノードが浮く
- Typography: ややシャープでおしゃれ
- Color: ネイビー、金、生成り
- Interaction: 星クリックでサイドパネルに全文

### Trade-offs
- Strong at: ワクワク感、ブランド感、探索感
- Weak at: 高齢者には少し抽象的に見える可能性

### Best for
- デモ映え、講座主への提案、未来感を見せたい場面
