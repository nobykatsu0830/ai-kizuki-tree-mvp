## Variant: テーマ群れマップ

### Design stance
AIで似た気づきを束ね、「学びの構造」と「次の問い」が見える実用寄りUIにする。

### Key choices
- Layout: 左サイドバー + テーマ別クラスター
- Typography: 読みやすさ優先
- Color: タグ別に淡い色分け
- Interaction: カードクリックで詳細ドロワー

### Trade-offs
- Strong at: 内容整理、講座主・事務局が使いやすい、AI活用が見える
- Weak at: 森/宇宙ほどの詩的な体験は弱い

### Best for
- 実運用、事務局レビュー、講座の振り返り・次回設計
