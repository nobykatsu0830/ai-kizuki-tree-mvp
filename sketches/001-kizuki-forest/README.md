## Variant: 気づきの森

### Design stance
高齢者にもやさしく、自然・余白・あたたかさで「感想を探索する」体験にする。

### Key choices
- Layout: 森の中にカードが点在するような自由配置
- Typography: 大きめ・読みやすい日本語
- Color: 生成り、緑、光の黄色
- Interaction: カードクリックで全文パネルを表示

### Trade-offs
- Strong at: やさしい、講座・セラピー文脈に合う、怖くない
- Weak at: グラフ感・構造感はやや弱い

### Best for
- 初期MVP、一般参加者、高齢者にも見せる公開ページ
